# QA Checklist

- Desktop, tablet, and mobile layouts verified.
- Keyboard navigation covers calculator forms, tabs, and actions.
- Inputs reject invalid negative values and impossible rates.
- Charts render for every calculator.
- Copy, share, print, PDF, and Excel actions are available.
- URL state restores form values after refresh.
- Swagger schema exposes all calculator endpoints.
- Lighthouse targets: Performance > 90, Accessibility > 95, SEO > 95, Best Practices > 95.
